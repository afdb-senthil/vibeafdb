# Project Summary
This project aims to create a digital platform similar to the African Development Bank (AfDB) website, focusing on institutional banking, development finance, and sustainable development across Africa. The platform provides resources and information related to projects, investments, and regional operations, ultimately promoting economic growth and development in African nations. Recently, the project has been containerized for deployment on Google Cloud, enhancing its scalability and manageability.

# Project Module Description
The project consists of several functional modules:
- **Header**: Contains navigation links and branding elements.
- **Hero**: Displays a rotating banner with key messages and call-to-action buttons.
- **Statistics**: Showcases impactful statistics about the bank's investments and reach.
- **Featured Projects**: Highlights major ongoing and upcoming projects.
- **News & Events**: Lists the latest news articles and upcoming events.
- **Topics & Countries**: Presents focus areas and regional operations.
- **Footer**: Provides contact information and additional links.

# Usage
To set up the project, follow these steps:
1. Install dependencies: `pnpm install`
2. Build the project: `pnpm run build`
3. Build the Docker container: `docker build -t afdb-website .`
4. Run the application locally: `docker run -p 8080:8080 afdb-website`
5. Push to Google Container Registry: 
   - `docker tag afdb-website gcr.io/YOUR_PROJECT_ID/afdb-website`
   - `docker push gcr.io/YOUR_PROJECT_ID/afdb-website`
6. Deploy to Cloud Run: 
   - `gcloud run deploy afdb-website --image gcr.io/YOUR_PROJECT_ID/afdb-website --platform managed --region us-central1 --allow-unauthenticated --port 8080`

For automated CI/CD, use:
- `gcloud builds submit --config cloudbuild.yaml`
