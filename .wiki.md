# Project Summary
This project aims to create a digital platform similar to the African Development Bank (AfDB) website, focusing on institutional banking, development finance, and sustainable development across Africa. The platform will provide resources and information related to projects, investments, and regional operations, ultimately promoting economic growth and development in African nations.

# Project Module Description
The project consists of several functional modules:
- **Header**: Contains navigation links and branding elements.
- **Hero**: Displays a rotating banner with key messages and call-to-action buttons.
- **Statistics**: Showcases impactful statistics about the bank's investments and reach.
- **Featured Projects**: Highlights major ongoing and upcoming projects.
- **News & Events**: Lists the latest news articles and upcoming events.
- **Topics & Countries**: Presents focus areas and regional operations.
- **Footer**: Provides contact information and additional links.

# File Description Inventory
- **README.md**: Project documentation and setup instructions.
- **components.json**: Configuration for components.
- **eslint.config.js**: ESLint configuration for code linting.
- **index.html**: Main HTML file for the application.
- **package.json**: Lists project dependencies and scripts.
- **postcss.config.js**: Configuration for PostCSS.
- **prerender/**: Contains scripts for pre-rendering blog routes and sitemaps.
- **public/**: Static assets such as favicons and robots.txt.
- **src/**: Main source directory containing components, hooks, pages, and libraries.
- **template_config.json**: Configuration for templates.
- **tsconfig.*.json**: TypeScript configuration files.
- **vite.config.ts**: Configuration file for Vite build tool.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development.
- **ESLint**: Tool for identifying and fixing problems in JavaScript code.

# Usage
To set up the project, follow these steps:
1. Install dependencies: `pnpm install`
2. Build the project: `pnpm run build`
3. Run the application: `pnpm run start`
